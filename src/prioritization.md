# MoSCoW Prioritization: Event App (MVP)

## MUST HAVE
- **User Authentication & Security**
    - Registration (including Date of Birth for age verification).
    - Login and Logout functionality.
    - Restricted access for guests (core features require authentication).
- **Core Map Experience**
    - Interactive Map with event pins.
    - Basic Event Card (Title, Time, Location).
- **Essential Actions**
    - Basic Create Event form (Title, Date, Description, Location, age restrictions, participant limits).
    - Core "Join / Unjoin" functionality.
- **Validation & Rules**
    - Automated enforcement of age restrictions.
    - Automated enforcement of participant limits.
- **Technical Foundation**
    - Responsive UI for both Mobile and Desktop.

## SHOULD HAVE
- **Discovery & Organization**
    - Filtering system (Category, City, Date).
    - "My Events" section for personal schedule management.
- **Communication**
    - Basic Email or Push notifications for event confirmations.
- **Advanced UX**
    - Password Recovery ("Forgot Password" flow).

## COULD HAVE
- **Advanced UX**
    - Time Overlap Prevention (preventing double-booking).
    - Map Pin Clustering (grouping nearby icons).
- **Performance & Offline**
    - Basic Offline Mode (viewing cached data when internet is lost).
    - Search by tags (e.g., #party, #education).

## WON'T HAVE
- **Social & Analytics**
    - Public Participant List (seeing other users' profiles).
    - In-app comment sections for events.
    - Detailed Analytics Dashboard for event organizers.
